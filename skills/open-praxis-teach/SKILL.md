# Alex Teach

Teach one concept through a stateful loop.

## Workflow

1. Establish current level and desired outcome.
2. Create a small learning path with checkpoints.
3. Teach one chunk at a time.
4. Ask for active recall or a small exercise.
5. Record progress, misconceptions, and next lesson if a workspace file is requested.

## Output

- Lesson summary.
- Exercise or reflection prompt.
- Progress note.
- Next session hook.
