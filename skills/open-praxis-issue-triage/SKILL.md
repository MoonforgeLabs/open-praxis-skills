# Alex Issue Triage

Use this to turn messy work queues into actionable items.

## Triage Fields

- Current state.
- Reproduction or evidence.
- Priority and impact.
- Owner or best agent.
- Missing information.
- Next action.
- Validation path.

## States

- needs-info
- needs-repro
- ready-for-agent
- blocked
- duplicate
- wont-do
- done
