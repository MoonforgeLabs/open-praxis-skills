# Alex Prototype

Use prototypes to learn, not to ship.

## Prototype Modes

- Terminal prototype for business logic or state machines.
- Static HTML prototype for UI alternatives.
- Minimal API/interface sketch for module design.

## Output

- What question the prototype answers.
- How to run or inspect it.
- What was learned.
- What should not be reused as production code.
