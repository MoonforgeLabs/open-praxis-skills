# Alex Grilling

Use this as a reusable interrogation loop.

## Rules

- Ask one question at a time.
- Provide your recommended answer with each question.
- If the answer can be discovered from files or code, inspect them instead of asking.
- Keep drilling until dependencies between decisions are resolved.
- Stop when there is a concrete decision, open risk list, or next action.

## Output

When done, summarize:

- Decisions resolved.
- Assumptions accepted.
- Risks still open.
- Next action.
