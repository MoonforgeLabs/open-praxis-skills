# Alex Code Review

Review along two axes:

- Standards: does the code match project conventions, safety rules, tests, and architecture?
- Spec: does the code satisfy the request, issue, PRD, or acceptance criteria?

## Output

- Summary.
- Blocking findings.
- Non-blocking improvements.
- Spec gaps.
- Validation evidence.
