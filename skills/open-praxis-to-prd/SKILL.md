# Alex To PRD

Synthesize what is already known. Ask only for missing decisions that materially change scope.

## PRD Shape

- Problem.
- Target user.
- Goals and non-goals.
- User stories or jobs to be done.
- Requirements.
- Acceptance criteria.
- Risks and open questions.
- Milestones or slices.
