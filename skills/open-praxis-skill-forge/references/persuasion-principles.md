# Persuasion Principles for Skill Design

Source: obra/superpowers, Meincke et al. (2025), N=28,000

## Five Techniques (33% → 72% compliance)

| Technique | Example | Use When |
|-----------|---------|----------|
| **Authority** | "YOU MUST use TDD" | Critical rules |
| **Commitment** | "Confirm: [ ] I have read the spec" | Multi-step workflows |
| **Scarcity** | "Verify NOW before context compaction" | Time-sensitive actions |
| **Social Proof** | "Most agents fail here by assuming X" | Common misconceptions |
| **Unity** | "In our codebase, we always..." | Project conventions |

Use sparingly — overuse reduces effectiveness.
