# Load Theory: Context vs Cognitive

Source: mattpocock/skills writing-great-skills

## Two Loads

- **Context Load** (model): description tokens in every session
- **Cognitive Load** (human): must remember which skills exist

## Tradeoff

More model-invoked ↑ context ↓ cognitive | More user-invoked ↓ context ↑ cognitive

Router skill: adds ~100 tokens context, eliminates cognitive load.

## Leading Word as Attention Anchor

Strong leading words (tracer bullet, relentless, fog of war) recruit pretraining priors.
Weak leading words (best practices, carefully) are no-ops → replace with stronger anchors.
